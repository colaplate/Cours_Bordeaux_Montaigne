MSM3Y2 // 2018-2019
====================================

## cours 01 - lundi 19 novembre 2018 - Introduction à p5.js
* [p5.js](https://www.p5js.org) + [éditeur p5.js](https://editor.p5js.org)
* révision des notions vues l'année précédente avec Processing : 
    * gestionnaires setup() et draw()
    * gestionnaires d'évènements mousePressed(), mouseDragged(), keyPressed().
    * fonctions de dessin, export avec la fonction *saveCanvas()*
    * variables.
* sketch dessin « stars » : https://editor.p5js.org/v3ga/sketches/BkWmrblRm
* création d'interface utilisateur avec la librairie [p5.dom.js](https://p5js.org/reference/#/libraries/p5.dom) 

### Références
* [Daniel Shiffman / Foundations in programming in Javascript - p5.js tutorial](https://www.youtube.com/playlist?list=PLRqwX-V7Uu6Zy51Q-x9tMWIv9cueOFTFA)

## cours 02 - lundi 26 novembre 2018
* Sketch plein écran
   * Référence : [Gildas Paubert - P5js-fullscreen](http://gildasp.fr/exp/P5js-fullscreen/)
* Boucles, pavage de l'espace avec deux boucles imbriquées.
* Modèle de particule : position, vitesse, contraintes sur les bords de la fenêtre. 

### Exercide à rendre pour le 5 janvier 2019
* Créer un mini-site web pour le prochain évènement (pour l'instant fictif) [GROW](https://www.grow.paris/fr/) en 2019 composé d'une page (pouvant scroller) comprenant :  
    * un fond animé interatif plein écran avec p5.js.
    * présentation d'au moins quatre œuvres our workshops ( image(s) + texte ) d'artistes qui figuraient à l'édition 2018.     

L'évaluation se fera : 
* sur la créativité de l'animation interactive en fond avec p5.js
* la qualité de la mise en page.
